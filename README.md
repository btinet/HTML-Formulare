## Einführung
HTML-Formulare dienen der Erfassung von Benutzereingaben und Interaktionen.
Ein Formular ist wie folgt aufgebaut:

````html
<form method="get">
    <label>
        Eingabefeld:
        <input type="text" name="input_field">
    </label>
    <button type="submit">Absenden</button>
</form>
````
